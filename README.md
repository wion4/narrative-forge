# Narrative Forge

AI-powered game narrative design toolkit for Claude Code. Write stories, dialogues, lore, and generate 2D concept art — all guided by a methodology distilled from 50+ masterworks.

## Installation

```bash
claude plugins marketplace add wion4/claude-marketplace
claude plugins install narrative-forge@wion4-marketplace
```

## Features

- **Story Architecture** — plot structures, act breakdowns, branching narratives
- **Dialogue Craft** — character voices, subtext, branching dialogue trees, ambient barks
- **Environmental Storytelling** — lore documents, item descriptions, audio logs, world-building
- **Character Design** — deep profiles with arcs, motivations, relationships, ensemble casts
- **Concept Art** — 2D visuals in SVG and HTML/Canvas (maps, characters, UI, symbols)

## Commands

```
/narrative-forge:story create     — new story from scratch
/narrative-forge:story dialogue   — write dialogue
/narrative-forge:story world      — build world/lore
/narrative-forge:story character  — design characters
/narrative-forge:story concept    — generate concept art
/narrative-forge:story review     — review existing narrative
```
